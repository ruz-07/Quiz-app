#  Quiz App

A simple and interactive **Quiz Application** built using **HTML, CSS, and JavaScript**. This project is designed to test users' knowledge through multiple-choice questions while providing an engaging user experience.

---

##  Features

-  Multiple Choice Questions
-  Real-time Score Calculation
-  Confetti Animation on Completion
-  Responsive Design
-  Fast and Lightweight
-  Clean and Modern UI
-  Restart Quiz Anytime

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
